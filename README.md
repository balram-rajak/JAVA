# JAVA
Java learning projects and hands on

## Projects

This repository contains various Java learning projects covering fundamental to intermediate concepts. Each project is in its own directory with a dedicated README file.

### 1. [HelloWorld](./HelloWorld)
The classic first program - prints "Hello, World!" to the console.
- **Concepts**: Basic syntax, main method, console output
- **Difficulty**: Beginner

### 2. [BasicCalculator](./BasicCalculator)
A console-based calculator for basic arithmetic operations.
- **Concepts**: User input, switch statements, arithmetic operations
- **Difficulty**: Beginner

### 3. [TemperatureConverter](./TemperatureConverter)
Converts temperatures between Celsius and Fahrenheit.
- **Concepts**: Mathematical calculations, conditionals, menu-driven programs
- **Difficulty**: Beginner

### 4. [StudentManagementSystem](./StudentManagementSystem)
A simple student management system demonstrating OOP concepts.
- **Concepts**: Classes, objects, encapsulation, constructors
- **Difficulty**: Intermediate

### 5. [BankingApplication](./BankingApplication)
Simulates basic banking operations with account management.
- **Concepts**: OOP, business logic, data validation, state management
- **Difficulty**: Intermediate

## How to Use

Each project directory contains:
- Java source files (`.java`)
- A README.md with detailed information about the project
- Instructions on how to compile and run the code

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/balram-rajak/JAVA.git
   cd JAVA
   ```

2. Navigate to any project directory:
   ```bash
   cd HelloWorld
   ```

3. Follow the instructions in the project's README.md file

## Prerequisites

- Java Development Kit (JDK) 8 or higher
- A text editor or IDE (Eclipse, IntelliJ IDEA, VS Code, etc.)
