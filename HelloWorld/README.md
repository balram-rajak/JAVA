# Hello World

## Description
A simple Java program that prints "Hello, World!" to the console. This is typically the first program written when learning a new programming language.

## Concepts Covered
- Basic Java syntax
- `main` method
- `System.out.println()` for console output

## How to Run
1. Compile the program:
   ```bash
   javac HelloWorld.java
   ```

2. Run the program:
   ```bash
   java HelloWorld
   ```

## Expected Output
```
Hello, World!
```

## Learning Objectives
- Understand the structure of a basic Java program
- Learn how to compile and run Java code
- Get familiar with Java's output statements
